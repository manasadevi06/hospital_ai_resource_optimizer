# AI Hospital Resource Optimizer

## 📌 Overview
This project implements an AI-driven hospital resource optimization system using Reinforcement Learning (PPO). It dynamically allocates beds, ICU units, and handles patient inflow in real-time.

## 🚀 Features
- Reinforcement Learning-based decision making
- Real-time simulation of hospital environment
- Interactive dashboard using Streamlit
- Optimized resource allocation

## 🧠 Technologies Used
- Python
- Reinforcement Learning (PPO)
- Stable-Baselines3
- OpenAI Gym
- Streamlit

## ▶️ How to Run

```bash
pip install -r requirements.txt
python model/train.py
streamlit run app/app.py
